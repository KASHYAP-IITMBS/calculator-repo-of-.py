# 🧮 Python Calculator

<br>

A professional command-line calculator built using Python.

<br>

This calculator performs mathematical operations, handles errors, 
supports continuous calculations, and saves calculation history.

<br>

---

## 🚀 Features

<br>

✅ Addition (+)

<br>

✅ Subtraction (-)

<br>

✅ Multiplication (*)

<br>

✅ Division (/)

<br>

✅ Supports expressions like `10*10/10`

<br>

✅ Unlimited calculations

<br>

✅ Type `exit` to close the calculator

<br>

✅ Error handling using try-except

<br>

✅ Saves calculation history automatically

<br>

## ▶️ How To Run

<br>

### 1. Clone Repository

<br>

```bash
git clone YOUR_REPOSITORY_LINK
```

<br>

### 2. Open Project Folder

<br>

```bash
cd calculator-repo-of-.py
```

<br>

### 3. Run Calculator

<br>

```bash
python calculator.py
```
